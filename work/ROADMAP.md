# Roadmap: Habit Tracker

## Overview

Build a fast, local-first habit tracker with gamified visuals. MVP focuses on quick check-ins, streak tracking, and motivational progress displays.

---

## Milestone 1: Foundation

**Goal**: Project setup and design system with bold, colorful aesthetic
**Target**: Week 1

### Phases

#### Phase 1.1: Project Setup
- Initialize Next.js 15 with TypeScript
- Configure Tailwind CSS with custom color palette (bold/gamified)
- Setup shadcn/ui components
- Configure Vitest for testing
- Setup PWA manifest and service worker basics

#### Phase 1.2: Design System
- Create design tokens (vibrant colors, playful typography)
- Build core components: Button, Card, Icon, Badge
- Create habit-specific components: HabitCard, StreakBadge
- Build design system showcase page
- User approval on visual foundation

#### Phase 1.3: Data Layer
- Setup Dexie.js for IndexedDB
- Define habit schema (id, name, icon, color, createdAt)
- Define completion schema (habitId, date, completedAt)
- Create Zustand store for UI state
- Build data hooks (useHabits, useCompletions)

### Success Criteria
- [ ] Dev server runs without errors
- [ ] Design system page shows all components
- [ ] Can persist and retrieve test data
- [ ] PWA installable on mobile
